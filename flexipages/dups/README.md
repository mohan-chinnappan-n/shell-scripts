# Script to look for duplicate Facet-Ids and Identifiers in Salesforce flexipages


## How to use?

```bash
Usage: fp_dup_finder.sh --input-folder <folder_path>
```

## Example

```bash
bash fp_dup_finder.sh --input-folder ~/tmp/fp       
```
